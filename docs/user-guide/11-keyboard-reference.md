# Keyboard Reference

## File

| Shortcut | Action |
|----------|--------|
| `Ctrl+N` | New song |
| `Ctrl+O` | Open project |
| `Ctrl+S` | Save |
| `Ctrl+Shift+S` | Save As |

## Playback

| Shortcut | Action |
|----------|--------|
| `Space` | Play / Pause |
| `Enter` | Play from cursor |
| `Escape` | Stop |

## Navigation

| Shortcut | Action |
|----------|--------|
| Arrow keys | Move cursor |
| `Tab` | Next channel |
| `Shift+Tab` | Previous channel |
| `Left`/`Right` | Wrap to adjacent channel at edge |

## Octave

| Shortcut | Action |
|----------|--------|
| `F1`-`F8` | Select octave 1-8 |
| `Page Up` | Octave up |
| `Page Down` | Octave down |
| `Ctrl+Up` | Octave up |
| `Ctrl+Down` | Octave down |

## Note Entry

```
 2  3     5  6  7
Q  W  E  R  T  Y  U     <- octave +1
C# D#    F# G# A#
 C  D  E  F  G  A  B

 S  D     G  H  J
Z  X  C  V  B  N  M     <- current octave
C# D#    F# G# A#
 C  D  E  F  G  A  B
```

| Shortcut | Action |
|----------|--------|
| `.` (period) | Insert rest |

## Editing

| Shortcut | Action |
|----------|--------|
| `Insert` | Insert empty row |
| `Delete` | Delete row at cursor |
| `Backspace` | Delete row and pull up |

## Selection and Clipboard

| Shortcut | Action |
|----------|--------|
| `Shift+Arrow` | Extend selection |
| `Ctrl+A` | Select all in pattern |
| `Ctrl+C` | Copy selection |
| `Ctrl+X` | Cut selection |
| `Ctrl+V` | Paste at cursor |

## Transposition

| Shortcut | Action |
|----------|--------|
| `=` or `+` | Transpose up 1 semitone |
| `-` | Transpose down 1 semitone |
| `Shift+=` | Transpose up 1 octave |
| `Shift+-` | Transpose down 1 octave |

## Instrument Entry

| Shortcut | Action |
|----------|--------|
| `0`-`9`, `A`-`F` | Enter hex digit (two digits for byte value) |
| `Escape` | Cancel pending hex digit |

## Undo / Redo

| Shortcut | Action |
|----------|--------|
| `Ctrl+Z` | Undo |
| `Ctrl+Y` | Redo |

## Channel Mute / Solo

| Shortcut | Action |
|----------|--------|
| Click header | Toggle mute |
| `Ctrl+Click` header | Toggle solo |
